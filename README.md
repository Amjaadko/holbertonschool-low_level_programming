# Pointers, arrays and strings

This project is part of the low level programming track at Holberton School.  
It focuses on learning about pointers, arrays, and strings in C.

## Requirements
- Allowed editors: vi, vim, emacs
- All files compiled on Ubuntu 20.04 LTS using gcc
- Code should follow the Betty style
- No use of the standard library (printf, puts, etc.)
- Allowed to use `_putchar`
- All function prototypes must be included in `main.h`

## Files
- `0-reset_to_98.c` - Function that takes a pointer to an int and updates the value it points to to 98.
- `main.h` - Header file containing function prototypes.

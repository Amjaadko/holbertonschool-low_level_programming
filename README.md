# Pointers, Arrays and Strings

This directory contains solutions to tasks related to pointers, arrays, and strings in C programming.

## Files

- **0-reset_to_98.c**: Function that takes a pointer to an int and updates the value it points to 98.
- **1-swap.c**: Function that swaps the values of two integers.
- **2-strlen.c**: Function that returns the length of a string.
- **3-puts.c**: Function that prints a string followed by a new line.
- **main.h**: Header file containing function prototypes.
- **README.md**: This file.

# Recursion - Holberton School

This directory contains exercises on **recursion** in C, part of the Holberton School curriculum.

## Files

- `_putchar.c` – Function to print a character to stdout.
- `0-puts_recursion.c` – Function `_puts_recursion` that prints a string followed by a new line using recursion.
- `0-main.c` – Test file for `_puts_recursion`.

## Compilation

To compile the code:

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 _putchar.c 0-puts_recursion.c 0-main.c -o 0-puts_recursion
